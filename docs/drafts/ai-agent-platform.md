#UI
let's create a simple agent launch platform that launches an agent and
  let's the user interact with the agent to perform a private payment with our existing program in a conversational interface that asks for an amount and destination and at the end
  provides the wallet address to deposit the funds and after that shows a loading
  indicator and the final result will be a link to solana devnet with the link of the
  redeem transaction.

#Infrastructure
Regarding the infrastructure we'll use vercel ai sdk to create the custom agent for the users. We'll work on the isolation of keypairs later
